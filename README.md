# Imers-o-Dados-Alura
# Análise Salarial Global na Área de Dados

## Sobre o Projeto

Este projeto consiste em um dashboard interativo desenvolvido para analisar salários de profissionais da área de dados ao redor do mundo.

A aplicação permite explorar informações sobre cargos, níveis de senioridade, tipos de contrato, porte das empresas e evolução salarial ao longo dos anos por meio de filtros dinâmicos e visualizações interativas.

## Objetivos

- Analisar tendências salariais na área de dados.
- Comparar remunerações entre diferentes cargos.
- Identificar padrões de trabalho remoto.
- Visualizar a evolução dos salários ao longo do tempo.
- Desenvolver habilidades em análise e visualização de dados.

## Tecnologias Utilizadas

- Python
- Pandas
- Streamlit
- Plotly

## Principais Funcionalidades

- Filtros por ano, senioridade, tipo de contrato e tamanho da empresa.
- Indicadores de desempenho (KPIs).
- Ranking de cargos por salário médio.
- Distribuição salarial.
- Análise de modalidades de trabalho.
- Mapa mundial com salários médios de Cientistas de Dados.
- Evolução salarial ao longo dos anos.
